üì¶ Installation & Setup

Step 1: Install Plugin

  1. Upload plugin folder to /wp-content/plugins/kirin-vulnerability-dashboard/
  2. Activate plugin in WordPress admin
  3. Go to Settings > Kirin VulnDB

Step 2: Configure API

  1. Set API Endpoint: http://your-server:8080/api
  2. Set API Key: wp-demo-key (or your custom key)
  3. Save settings

Step 3: Use Shortcodes

Add shortcodes to any post, page, or widget area:

[kirin-vulnerabilities limit="10" theme="light"]
